# Contributing

Thank you for your interest in contributing to claude-switch!

## Development Setup

1. Clone the repository:

```bash
git clone https://github.com/jigonr/claude-switch.git
cd claude-switch
```

2. Install dependencies:

```bash
bun install
```

3. Run in development mode:

```bash
bun run dev
```

## Code Quality

### Linting

```bash
bun run lint
```

### Type Checking

```bash
bun run typecheck
```

### Testing

```bash
bun run test
```

## Commit Guidelines

We use [Conventional Commits](https://www.conventionalcommits.org/):

- `feat:` - New features
- `fix:` - Bug fixes
- `docs:` - Documentation changes
- `chore:` - Maintenance tasks
- `refactor:` - Code refactoring
- `test:` - Test changes

## Pull Requests

1. Fork the repository
2. Create a feature branch: `git checkout -b feat/my-feature`
3. Make your changes
4. Run tests and linting
5. Commit with conventional commit message
6. Push and create a PR

## Reporting Issues

Please use [GitHub Issues](https://github.com/jigonr/claude-switch/issues) to report bugs or request features.
