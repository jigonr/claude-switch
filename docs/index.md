# claude-switch

Simple API provider switcher for Claude Code CLI.

Switch between `claude-pro-max`, `anthropic`, and `z.ai` providers with a single command.

## Quick Start

```bash
# Install
bun add -g @jigonr/claude-switch

# Switch providers
claude-switch anthropic
```
